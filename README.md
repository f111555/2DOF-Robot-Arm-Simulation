# 2-DOF Robot Arm Simulation (Python)

本项目是一个基于 Python 开发的双自由度机械臂运动学仿真系统，旨在演示机器人基础理论在实际编程中的应用。

## 🚀 功能特性
* **正/逆运动学求解**：实现了 2-DOF 平面机械臂的几何解析法。
* **轨迹规划**：支持生成圆形、心形等自定义笛卡尔空间轨迹。
* **实时动态显示**：利用 Matplotlib 实现机械臂运动过程的交互式可视化。
* **异常处理**：具备工作空间边界检查功能，防止无效解导致程序崩溃。
* **双肘部姿态**逆运动学多解可视化：支持肘部向上（Elbow-up）和肘部向下（Elbow-down）两种姿态的实时对比。

## 🛠️ 技术栈
* 语言：Python 3.x
* 库：NumPy (数值计算), Matplotlib (数据可视化)

## 📂 项目结构
* `src/`: 核心算法及主程序
* `docs/`: 存放数学推导笔记及演示动图

## 📝 快速开始
1. 安装依赖：`pip install numpy matplotlib`
2. 运行程序：`python src/main.py`
